VUE项目创建

创建vue项目

作为练习创建vue项目时 vue create demo

手动为项目配置的时候，不要选择 linter/Formatter, 格式化程序，不然会在程序中报很多格式错误

项目准备

vue-cli脚手架

  1. 全局安装vue-cli（一台电脑只需要安装一次）

    npm i vue-cli -g

  2. 使用vue-cli创建vue项目

    vue create 项目名称

项目的基本配置

在vue项目中可以创建一个文件 vue.config.js

  1. 配置浏览器打开窗口的域名、端口号
  2. 启动项目时是否自动打开窗口

axios的封装与挂载

  1. 安装

    npm i axios

  2. 在main.js文件中引入

    import axios from 'axios'

  3. 在 main.js 中配置请求的基准路径

    axios.default.baseURL = '路径'

  4. 将 axios 挂载到 Vue 原型对象上（$http 只是自定义的名称而已）

    Vue.prototype.$http = axios

    在组件中使用 axios：this.$http

项目结构

在开发项目时：

  1. 真正需要展示在页面中的组件一般放在 文件夹components 目录下
  2. 在单独实现组件功能的时需要测试查看页面效果，因此还需要一个测试用的组件，一般放在 文件夹view 目录下
  3. 单独组件的展示效果可以通过路由设置路由规则，访问指定的路径可以查看到组件页面效果
