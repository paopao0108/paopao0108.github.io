Koa2

Koa2

特点

  • 支持async/await

  • 洋葱模型的中间件

Koa2基本使用

  1. 检查Node的环境

    Koa2 的使用要求 Node 版本在7.6及以上，通过 node -v 检查node的版本

  2. 安装Koa

    • npm init -y 创建package.json 文件，用于维护第三方包的信息

    • npm i koa 下载最新版本的koa到当前项目中

      下载速度慢的话，可以通过淘宝进行下载

  3. 创建并编写app.js文件

    • 创建Koa实例对象

    • 编写响应函数（中间件）

      • ctx 上下文，web容器，ctx.request 可以拿到请求对象 ctx.response可以拿到响应对象
      • next 下一个中间件，下一层中间件是否能够得到执行，取决于next这个函数有没有被调用

    • 绑定端口号 3000

      通过 app.listen 就可以指明一个端口号

  4. 启动服务器

    通过 node app.js 就可以启动服务器了

随即打开浏览器, 在浏览器中输入 127.0.0.1:3000/ 你将会看到浏览器中出现 hello world 的字符 串, 并且在服务器的终端中, 也能看到请求的 url

中间件特点

  • Koa对象通过use方法加入一个中间件
  • 一个中间件就是一个函数
  • 中间件的执行顺序符合洋葱模型
  • 内层中间件能否篇执行取决于外层中间件的next函数是否调用
  • 调用next函数得到的是Promise对象

执行结果

通过next函数进入下一个中间件（下一个函数的调用），但是next并不是简单的下一个函数的调用，而是其返回值外包裹一层promise，返回的是Promise对象

若想要拿到promise里面的内容，可以使用 async await
