axios

Axios 是一个基于 promise 网络请求库，作用于node.js和浏览器中

Axios 是专注于网络数据请求的库。
相比于原生的 XMLHttpRequest 对象，axios 简单易用。
相比于 jQuery，axios 更加轻量化，只专注于网络数据请求

axios一个专注于请求的库

注意：axios()方法返回的是一个 Promise 实例

npm install axios

注意：属性 params 与 data 二者选一，根据发的是 GET 请求 还是POST请求决定

await

await 表达式会暂停当前 async function 的执行，等待 Promise 处理完成。

  • 若 Promise 正常处理 (fulfilled)，其回调的 resolve 函数参数作为 await 表达式的值，继续执行 async function。
  • 若 Promise 处理异常 (rejected)，await 表达式会把 Promise 的异常原因抛出。

若 Promise 处理异常 (rejected)，await 表达式会把 Promise 的异常原因抛出。

另外，如果 await 操作符后的表达式的值不是一个 Promise，则返回该值本身。

某个方法的返回值是 Promise 实例，则前面可以添加 await 但 await 只能用在被 async 修饰的方法中,
