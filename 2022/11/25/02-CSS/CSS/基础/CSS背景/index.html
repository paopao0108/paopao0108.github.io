CSS背景

背景属性可以设置背景颜色、背景图片、背景平铺、背景图片位置、背景图像固定等

背景颜色

backgroun-color属性定义了元素的背景颜色

一般情况下元素背景颜色默认值为：transparent(透明)

背景色半透明

  • 最后一个参数时alpha透明度，取值范围在0~1之间
  • 习惯把透明度小数前的0去掉，写为background: rgba(0,0,0,.3);
  • 注意，背景半透明指的是盒子背景半透明，盒子里面的内容的不受影响

背景图片

background-image属性描述了元素的背景图像，实际开发中常见于logo或者一些装饰小图片、超大背景图等，优点电视非常便于控制位置

参数值 作用
none 无背景图（默认）
url 使用绝对或者相对地址指定背景图像，url里面的路径不需要加引号

注意：属性值url()括号里面写路径，url不能省！

背景平铺

background-repeat

参数值 作用
repeat 背景图像可以在纵向和横向上平铺
no-repeat 背景图像不平铺
repeat-x 在横向上平铺
repeat-y 在纵向上平铺

注意：页面既可以添加背景颜色，也可以添加背景图片，不过背景图片会压着背景颜色

背景图片的位置

background-position 属性可以改变图片在背景中的位置

参数值 说明
length 百分数 | 由浮点数和单位标识符组成的长度值
position top | bottom | left | center | right 方位名词

参数是方位名词

  • 若指定的两个都是方位名词，则两个值前后顺序无关，如 left top 和 top left效果一致
  • 若之制定了一个方位名词，另一个值省略，则第二个默认居中对齐

参数是精确单位

  • 若参数值是精确坐标，那么第一个一定是x坐标，第二个一定是y坐标
  • 若只指定一个数值，那么该数值一定是x坐标，另一个默认居中

参数是混合单位

  • 若指定的两个值是精确单位和方位名词混合使用，第一个还是x坐标，第二个值是y坐标

背景固定

background-attachment 属性设置背景图像是否固定或者随着页面的其余部分滚动

可以制作视差滚动的效果

参数 作用
scroll 背景图像是随对象内容滚动
fixed 背景图像固定

背景延伸

background-clip 设置元素的背景（背景图片或颜色）是否延伸到边框、内边距盒子、内容盒子下面

参数 说明
border-box 背景延伸至边框外沿
padding-box 背景延伸至内边距padding外沿
conten-box 背景被裁剪至内容区外沿
text 背景被裁剪成文字的前景色

背景属性的复合写法

为了简化属性的代码，可以将这些背景属性合并简写在同一个属性background中。从而节约代码量。

使用简写时，没有特定的书写顺序，一般习惯性约定顺序为：

background: 背景颜色 背景图片地址 背景平铺 背景图像滚动 背景图片位置
