元素显示模式

元素显示模式

什么是元素显示模式

作用：网页的标签非常多，在不同地方会用到不同类型的标签，了解他们的特点可以更好地布局我们的页面

元素显示模式就是元素（标签）以什么方式进行显示，比如<div>自己独占一行，比如一行可以放多个<span>.

HTML元素一般分为块元素和行元素

块元素

常见的块元素有：

  • <h1>~<h6>
  • <p>
  • <div>
  • <ul> <ol> <li>

其中<div>是最典型的块元素

块元素特点：

  • 独占一行
  • 高度、宽度、外边距以及内边距都可以控制
  • 宽度默认是容器（父级宽度）的100%
  • 是一个容器及盒子，里面可以放行内或者块级元素

注意：

  • 文字类的元素不能使用块级元素
  • <p>标签主要用于存放文字，因此<p>里面不能放块级元素，特别是不能放<div>
  • 同理，<h1>~<h6>等都是文字类块级标签，里面也不能放其他块级元素

行内元素

常见的行内元素有：

  • <a>
  • <strong> <b> <em> <i> <del> <s> <ins> <u>
  • <span>

其中<span>标签是最典型的行内元素，也叫内联元素

行内元素的特点：

  • 相邻行内元素在一行上，一行可以显示多个
  • 高、宽直接设置是无效的
  • 默认宽度就是它本身内容的高度
  • 行内元素只能容纳文本或者其他行内元素

注意：

  • 链接里面不能再放链接
  • 特殊情况链接<a>里面可以放块级元素，但是给<a>转换以下会计模式更加安全。

行内块元素

常见的行内块元素有：

  • <img />
  • <input />
  • <td>

它们同时具有块元素和行元素的特点：

  • 和相邻行内元素(行内块)在一行上，但是他们之间会有空白缝隙，一行可以显示多个（行内元素特点）
  • 默认宽度就是它本身内容的宽度
  • 高度、宽度、外边距以及内边距可以控制

元素显示模式转换

当想要增加<a>的触发范围时，就需要将进行模式转换

转换方法：

属性：属性值 作用
display: block 转换为块元素
display: inline 转换为行元素
display: inline-block 转换为行内块元素
