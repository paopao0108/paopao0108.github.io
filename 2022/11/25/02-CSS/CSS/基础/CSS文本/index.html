CSS文本

CSS中的text属性可以定义文本的外观

文本颜色

color属性用于送一文本的颜色

color属性的属性值表示方式：

表示方式 属性值
预定义的颜色值 red, green, blue,等
十六进制 #FF0000, #0000FF, #00FF00等
RGB代码 rgb(255,0,0)或rgb(100%,0%,0%)
  • 常用十六进制

文本对齐

text-align属性用于设置元素内文本内容的水平对齐方式

属性值 解释
left 左对齐（默认值）
right 右对齐
center 居中对齐

文本装饰

text-decoration属性规定添加文本的修饰，可以给文本添加下划线、删除线、上划线等

属性值 描述
none 默认。没有装饰线
underline 下划线。链接a自带下划线
overline 上划线。
line-through 删除线

取消链接标签a的下划线

文本缩进

text-indent属性用来指定文本的第一行的缩进，通常是将段落的首行缩进

语法

表示缩进10px，所有段落的第一行都可以缩进一个给定的长度，甚至这个长度可以设置为负值

em是一个相对单位，就是当前元素（font-size）1个文字的大小，如果当前元素没有设置大小，则会按照父元素的1个文字大小来进行缩进

行间距

line-height属性舒勇设置行间的距离（行高），可以控制文字行与行之间的距离。

行间距 = 文本高度+上间距+下间距（上下间距是相等的）

文字阴影(CSS3)

CSS3中，使用text-shadow属性将阴影应用于文本

值 属性
h-shadow 必需，水平阴影的位置，允许负值
v-shadow 必需。垂直阴影的位置，允许负值
blur 可选。模糊距离
color 可选。阴影的颜色。（参阅CSS颜色值）

文本属性总结

属性 表示 注意点
color 文本颜色 常用十六进制
text-align 文本对齐 设定文本水平对齐方式
text-indent 文本缩进 用于段落首行缩进2个字的距离 text-indent: 2em
text-decoration 文本装饰 下划线underline、上划线、删除线、空none
line-height 行高 控制行与行之间的距离
text-shadow 文本阴影

行内块与文字居中对齐

vertical-align：经常用于设置图片或者表单（行内块元素）和文字垂直对齐。

即用于设置一个元素的垂直对齐方式，但是它只针对与行内元素或者行内块元素有效

语法：

值 描述
baseline 默认，元素放置在父元素的基线上
top 把元素的顶端与行中最高元素的顶端对齐
middle 把此元素放置在父元素的中部
bottom 把元素的顶端与行中最低元素的顶端对齐

如右所示：

可以利用vertical-align对图片、表单等行内块元素进行设置，从而达到图片、表单与文字垂直居中对齐的效果

解决图片底部缝隙

由于行内块元素默认与文字的基线baseline对齐，故图片底部会有一个空白缝隙。

解决方法：

  1. 给图片添加vertical-align: middle | top | bottom 等（提倡使用）
  2. 把图片转换成块级元素，display: block;

溢出文字省略号显示

单行文本

单行文本溢出显示省略号，需满足以下条件：

  1. 先强制一行内显示文本

    white-space: nowrap; (默认normal自动换行)

  2. 超出部分隐藏

    overflow: hidden;

  3. 文字用省略号替代超出的部分

    text-overflow: ellipsis;

多行文本

多行文本溢出显示，有兼容性问题，适合于wedKit浏览器或者移动端
