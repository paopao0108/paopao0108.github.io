09 VUE组件

vue组件

什么是组件化开发

根据封装的思想,把页面上可重用的UI结构封装为组件,从而方便项目开发和维护

组件的后缀名是 .vue 上面介绍的App.vue就是一个组件

vue组件的组成部分

  • template 组件的模板结构
  • script 组件的JavaScript行为
  • style 组件的样式

每个组件必须包含template模板结构,script和style是可选的

template

vue2中template只能包含唯一的根节点

script

vue规定:开发者可以在script标签节点中封装组件的 javascript 业务逻辑

注意:

vue组件中的data必须是一个函数,不能直接指向一个数据对象

style

组件内的style节点是可选的

组件间的关系

直接封装好的组件之间是相互独立的,如: App.vue, Left.vue, right.vue

父子关系

在App.vue组件中使用Left.vue组件, 就形成了父子关系.步骤如下:

  1. 使用 import 导入需要的组件

  2. 使用 components 节点注册组件

    如果在“声明组件”的时候，没有为组件指定 name 名称，则组件的名称默认就是“注册时候的名称”

  3. 以 标签 的形式使用刚才注册的组件
