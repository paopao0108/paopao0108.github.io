    文件夹中会自动新建一个dist文件夹以及bundle.js文件

    在bundle.js文件中就有src/index.js中的内容

  5. 新建文件 src/index.html

  6. 安装插件 html-webpack-plugin 用于解析 html 文件的插件

  7. 安装插件 webpack-dev-server 用于启动服务的插件

  8. 安装完后，再去webpack配置文件中添加配置

    • 配置插件，在plugins选项中配置
      • template是要解析的html文件
      • filename是要产出的html文件

    • 配置devServer，用于启动本地服务

      • port：本地服务的端口号
      • directory：当前的目录（就是出口dist）

  9. 配置完webpack，还需要再package.json中添加

  10. 在命令行运行 npm run dev

    在浏览器中输入： http://localhost:3000 或 http://localhost:3000/index.html 即可访问 index.html中的内容

  11. 若运行 npm run build

    则在出口文件夹dist下，还会有一个index.html 文件
