grid网格布局

grid网格布局

display: grid

常见属性

属性 属性值
grid-template-columns 定义每一列的列宽，如 100px 100px 100px; 或 repeat(3,100px); 总共三列，每列列宽是100px
grid-template-rows 定义每一行的行高，如 100px 100px 100px; 或 repeat(3,100px);从上至下，每行高度为100px

grid-template-columns的属性值

使用px

grid-template-columns: 100px, 100px 80px;

使用repeat

  • repeat(次数, 大小)：grid-template-columns: repeat(2, 100px 80px); 等价于 grid-template-columns: 100px 80px 100px 80px;

  • repeat(auto-fill, minmax(50px, 1fr))

    • minmax()表示一个范围，第一个参数为最小数，第二个参数为最大数

      • 屏幕比较小时，每个格子都是50px，屏幕比较大时，每个格子的宽度=容器宽度/格子数量）-gap，即将容器宽度等分给每个格子

    • auto-fill是尽量填充格子的模式

      • 举例：

        格子最小为50px时，7个格子为 7*50=350，8个格子为 8*50=400， 9个格子为 9*50=450

        若屏幕宽度为 [350, 400]会显示7个格子，每个格子宽度为 width/7

        若屏幕宽度为 [400, 450] 会显示8个格子，每个格子宽度为 width/8

  • repeat(auto-fit, minmax(50px, 1fr))

    • auto-fit不会填充没有元素的格子，当屏幕较大时，只会给已有的元素等分宽度
使用fr

  • 单位fr：grid-template-columns: 1fr 2fr; 表示 后者列宽是前者的两倍

使用fr和px配合

  • 单位fr也可以和px配合使用：grid-template-columns: 800px 1fr 2fr; 第1列列宽800px，第3列列宽为第2列的2倍

使用minmax()

  • grid-template-columns: 1fr 5fr minmax(100px, 2fr); 第一列是1fr，第二列是5fr，第三列最小值是100px,最大是2fr。当第二列fr无限大时和第三列到100px时，会往第一列借值

使用auto
  • auto自适应：grid-template-columns: 100px auto 100px;
定义网线格名称

指定每一根网格线的名字，方便以后的引用。也可以有多个名字；[c1,c1a]

grid-template-rows

间距

  • row-gap:行间距

  • colum-gap:列间距

  • row-gap和column-gap简写形式: gap:20px 10px;

  • gap:20px; 等价于 行间距、列间距都为20px

grid-template-areas

网格布局允许指定”区域”（area），一个区域由单个或多个单元格组成。grid-template-areas属性用于定义区域。

可用于合并单元格　　　

grid-auto-flow

划分网格以后，容器的子元素会按照顺序，自动放置在每一个网格。默认的放置顺序是”先行后列”，即先填满第一行，再开始放入第二行；

默认是row；

参数形式：

  • 单个关键字：row、column，或 dense 中的一个。
  • 两个关键字：row dense 或 column dense。

justify-items align-items place-items

justify-items设置单元格水平位置：设置单元格内容的水平位置（左中右）；

align-items设置单元格垂直位置：设置单元格内容的垂直位置（上中下）；

二者的属性值可以取：

  • start：对齐单元格的起始边缘。
  • end：对齐单元格的结束边缘。
  • center：单元格内部居中。
  • stretch：拉伸，占满单元格的整个宽度（默认值）。

二者的简写为：

  • place-items: align-items justify-items

justify-content align-content place-content

justify-content：整个内容区域在容器里面的水平位置（左中右）；

align-content：整个内容区域的垂直位置（上中下）。

二者简写为：

place-content: align-content justify-content;
