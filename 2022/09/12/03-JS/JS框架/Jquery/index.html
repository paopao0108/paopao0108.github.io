JQuery对象

  1. $ 是 JQuery 的别称,在代码中可以用JQuery来代替 $ (一般使用$)

  2. JQuery的顶级对象是 $ ，相当于原生JavaScript的window，把元素利用$包装成JQuery，就可以调用JQuery的方法

JQuery对象 & DOM对象

  • JQuery对象只能使用JQuery的方法，DOM对象只能使用原生的JavaScript的属性和方法

  • DOM对象与JQuery对象的相互转换

    • DOM对象转为Jquery对象：

      $(DOM对象)

    • Jquery对象转为DOM对象

      方式1：$(‘div’)[index]

      方式2：$(‘div’).get(index)

Jquery样式操作

操作 css 方法

  1. 参数只写属性名，返回属性值

    $(this).css(‘color’); // 返回this中字体的颜色值

    $(this).css(‘width’); // 返回宽度，且有单位

    注意：返回的宽度有单位

  2. 参数写：属性名，属性值，逗号分隔，是设置一组样式，属性必须加引号，值若是数字可以不用跟单位和引号

    $(this).css(‘color’, ‘red’) // 将this中字体的颜色设置为红色

  3. 参数可以是对象形式，方便设置多组样式，属性名和属性值用冒号隔开，属性可以不用加引号

    $(‘div’).css({

    ​ ‘width’: ‘200px’,

    ​ ‘height’: 200,

    ​ ‘color’: ‘black’,

    ​ ‘background’: ‘pink’

    });

遍历元素

jquery隐式迭代是对同一类元素做同样的操作，若需要给同一类元素做不同操作，就需要用到遍历

语法1：

$(‘div’).each(function(index, domElem) {xxx;})

  • each() 遍历匹配每一个元素。主要用DOM处理
  • 里面的回调函数有2个参数：index是每个元素的索引号，domElem是每个DOM元素对象，不是jquery对象
  • 要想使用jquery方法，需要给这个dom元素转换为jquery对象，即$(‘domElem’).

语法2：

$.each(object, function(index, elem) {xxx;})

  • $.each() 可用于遍历任何对象，主要用于数据处理，比如 数组、对象
  • 里面的函数有2个参数：index是每个元素的索引号，elem遍历内容

内部添加

设置获取元素偏移

offset()

  • 设置或返回被选元素相对于 文档 的偏移坐标，跟父级没有关系
  • 该方法有两个属性：left 和 top。
    • offset().left 用于获取距离文档左侧的距离
    • offset().top 用于获取距离文档顶部的距离
  • 可以设置元素的偏移：offset({top: 10, left: 30})

position()

  • 获取距离带有定位父级位置的偏移坐标
  • 若无定位，以文档为准
  • 此方法不能设置偏移

scrollTop()/scrollLeft()

  • 可以获取、设置被卷去的头部(或 左侧)

单个事件注册

与原生基本一致: mouseover mouseout blur focus change keydown keyuop resize scroll

事件注册on()

  • on可以绑定多个事件，多个处理事件处理程序

    $(‘div’).on({

    })

  • 若事件处理程序相同

    $(‘div’).on(“mouseover mouseout”, function () { 事件处理程序; });

    即事件绑定在ul上，但通过点击li来触发，经过冒泡后执行alert

  • (重要)on可以给未来动态创建的元素绑定事件

jQuery事件处理

一次触发one()

只能触发一次，触发后自动解绑

$(‘div’).one(‘click’, function () {事件处理程序;});

自动触发trigger()

有些事件需要自动触发的，可以结合定时器来自动触发按钮点击事件等，就不必鼠标点击触发了

  1. 简写形式

    elem.click()

  2. 自动触发trigger()

    elem.trigger(‘事件’)

  3. 自动触发triggerHandler()，不会触发元素的默认行为

    如：获得input光标后，不会触发input的光标闪动

    elem.triggerHanfler(‘事件’)

示例：

// 自动触发,无需点击鼠标

// 方式1

// 方式2

$(‘p’).click();

// 方式3

$(‘p’).triggerHandler(‘click’);

jQuery其他方法

jQuery拷贝对象

若想要把某个对象拷贝（合并）给另外一个对象使用，此时可以使用$.extend()方法

语法：

$.extend([deep], target, object1, [objectN]);

  • deep: 为true时是深拷贝，默认为false 浅拷贝
  • target：要拷贝的对象
  • object1：待拷贝到第一个对象的对象
  • objectN: 待拷贝到第N个对象的对象
  • 浅拷贝是把被拷贝对象复杂数据类型中的地址拷贝给目标对象，修改目标对象会影响被拷贝对象（简单数据类型不会被影响）
  • 深拷贝是完全克隆，拷贝的对象，而不是地址，修改目标对象不会影响被拷贝对象

多库共存

jQuery使用$作为标识符，，其他js库可能也会用$作为标识符，这样一起使用就会引起冲突

为了解决这个问题，让jQuery和其它的js库不冲突，可同时存在，这就叫多库共存

解决方案：

  1. 将里面的$符号统一修改为jQuery，如jQuery(‘div’)

  2. 让jquery释放对 $ 的控制权，让用户自己决定

    var newJq = jQuery.noConflict(); // 此时newJq就是原来的 $
